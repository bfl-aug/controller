##操作服务器
___
功能： \
1.保存服务器信息 \
2.获取已保存的服务器 \
3.删除服务器 \
4.多服务器/单服务器 进行搜索文件 \
5.查看文件详情 \
6.下载文件 \
7.上传文件/可一次上传至多个服务器 \
8.删除文件 \
9.多服务器/单服务器 运行 相同命令 \
10.多服务器/单服务器 运行 不同命令

---
接口文档 \
    http://127.0.0.1:8181/docs \
    http://127.0.0.1:8181/redoc \
可选择是否开启接口文档
在main.py中找到docs_url=None，redoc_url=None进行控制
